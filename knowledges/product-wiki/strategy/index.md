# 产品策略 — LLM Translator

## 愿景

让任何人都能用上最好的大模型翻译，且完全掌控模型来源与数据去向。

## 价值主张

| 用户 | 痛点 | 价值 |
|------|------|------|
| 普通阅读用户 | 传统机翻生硬、不懂上下文 | LLM 语义级翻译，译文更自然 |
| 技术用户 | 现有翻译插件模型固定、不可控 | 自带 Key、接本地模型、Prompt 可定制 |

## 关键能力

- LLM 提供方配置（云端 OpenAI 兼容 + 本地 Ollama）——**不内置任何模型接口**，仅支持用户配置自定义接口
- 划词翻译：划词后出现浮动触发按钮，点击翻译
- 目标语言默认浏览器首选语言，可覆盖
- Prompt 自定义

## 增长

- 第一版不涉及商业化，暂不定价（见「商业模式：暂不定价」）。
- 增长依赖口碑与技术社区传播（自带 Key 用户群体）。

## 防御性

- 模型无关的适配层设计，新模型快速接入。
- 隐私可控定位形成用户信任壁垒。

## 商业模式

**暂不定价**。第一版为开源 / 免费，聚焦验证核心价值与翻译体验。商业化延后评估。
