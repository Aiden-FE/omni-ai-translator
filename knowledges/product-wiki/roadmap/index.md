# 路线图 — LLM Translator

## v0.1 — 划词翻译 MVP（第一版，立项范围）

- [x] MV3 + WXT + Vue 3 + TS 骨架
- [x] LLM 提供方配置（云端 OpenAI 兼容 + 本地 Ollama，不内置模型）
- [x] 划词翻译：浮动触发按钮 + 点击翻译 + 浮层展示
- [x] 目标语言默认浏览器首选语言，可覆盖
- [x] 连通性测试

## v0.2 — 翻译源配置闭环（进行中）

- [ ] 统一翻译源适配层（LLM + 传统翻译源 provider 抽象与路由）
- [ ] 免 Key 兜底翻译源（内置 Google + 微软免 Key 公共端点，开箱即用）
- [ ] 传统 API Key 配置（Google/微软填自有 Key，覆盖兜底）
- [ ] 配置页翻译源选择 UI（源类型选择、生效源展示、连通性测试扩展）

> 本轮调整 v0.1「不内置任何模型接口」原则：内置传统翻译免 Key 公共端点作为兜底。隐私声明相应更新。

## v0.3 — 小窗输入翻译（由 v0.2 原「小窗输入」顺延）

- [ ] 独立输入翻译窗口
- [ ] 历史记录

## v0.4 — 全文翻译（由 v0.3 顺延）

- [ ] 整页 DOM 改造与样式隔离
- [ ] 分段翻译与并发控制

## 后续评估

- 跨浏览器（Firefox / Edge）
- 商店上架
- 商业化与付费方案
