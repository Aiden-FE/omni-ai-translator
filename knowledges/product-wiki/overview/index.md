# 产品概览 — LLM Translator

## 一句话定位

一个基于 LLM 的 Chrome 浏览器翻译插件，支持用户配置云端或本地大模型接口，提供高质量翻译。

## 核心价值

- **翻译质量**：借助 LLM 提供优于传统机翻的译文，理解上下文与语义。
- **模型自选**：用户可配置云端大模型（OpenAI 兼容）或本地大模型（Ollama 等），兼顾质量、成本与隐私。
- **隐私可控**：API Key 仅存本地，本地模型方案数据不出本机。

## 目标用户

- 普通阅读用户：开箱即用，期望更高质量的 AI 翻译。
- 技术用户 / 开发者：自带 Key、接入本地模型、深度自定义 Prompt。

## 核心场景

1. 划词翻译（第一版）：选中文本即译，浮层展示。
2. 全文翻译（规划中）：整页翻译。
3. 小窗输入翻译（规划中）：独立输入框翻译。
