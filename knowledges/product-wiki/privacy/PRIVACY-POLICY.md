# 隐私政策 — LLM Translator

> 最后更新：2026-06-30
> 本插件高度重视用户隐私。本政策说明数据如何收集、使用与存储。

## 1. 我们收集什么

本插件**不收集**任何个人身份信息，不内置任何分析或追踪 SDK。

## 2. 你提供的数据

| 数据类型 | 用途 | 存储位置 |
|----------|------|----------|
| LLM API Key / BaseURL | 调用你配置的大模型接口 | 仅本地 `chrome.storage.local`，不上传 |
| 选中的待翻译文本 | 发送给你配置的 LLM 接口完成翻译 | 不在本插件留存，仅按需透传给 LLM 接口 |
| 插件设置（目标语言、Prompt 等） | 记忆你的偏好 | 仅本地 `chrome.storage.local` |

## 3. 数据流向

- 待翻译文本**仅**发送给你当前启用的 LLM 提供方（云端或本地）。
- 若你启用本地模型（如 Ollama），文本仅在本机流转，不外传。
- 若你启用云端模型，文本将按该云服务商的隐私政策处理，**请自行评估其隐私条款**。
- 本插件**不会**将你的文本或 Key 发送给插件作者或任何第三方。

## 4. 数据安全

- API Key 仅存于浏览器本地存储，不出现在日志、错误上报或 commit 中。
- 建议用户妥善保管 Key，定期轮换。

## 5. 第三方服务

本插件本身不依赖第三方服务；翻译能力由你配置的 LLM 提供方提供，其隐私政策独立适用。

## 6. 数据删除

你可在浏览器扩展管理中卸载本插件，或在本插件选项页清除配置，所有本地数据随即删除。

## 7. 合规说明

- 不涉及 Cookies、广告追踪。
- 不面向未成年人收集数据。
- 涉及云端传输时，数据出境与处理责任由用户选择的云服务商承担。

## 8. 联系与变更

本政策随版本更新可能调整，更新后将在本文件标注日期。法律复核建议在正式发布前进行。
