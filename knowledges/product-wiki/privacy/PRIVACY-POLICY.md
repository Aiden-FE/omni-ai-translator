# 隐私政策 — LLM Translator

> 最后更新：2026-07-03
> 本插件高度重视用户隐私。本政策说明数据如何收集、使用、共享与存储。
> **本文仅供信息参考，不构成法律建议。正式发布前必须由具备数据隐私法律资质的律师复核。**

## 前言

LLM Translator 是一款基于 LLM 的 Chrome 浏览器翻译插件，允许用户配置翻译源（云端大模型、本地大模型、传统翻译接口），提供划词翻译等能力。本政策说明插件如何处理你的数据。

如对本政策有疑问，可通过项目仓库（GitHub: Aiden-FE/llm-translator）提 issue 联系。正式隐私联系邮箱与责任主体信息待补充 `[⚠️ 需要法律复核]`。

## 1. 我们收集的信息

本插件**不收集**任何个人身份信息，不内置任何分析或追踪 SDK，不使用 Cookie 追踪用户行为。

插件仅在本地处理以下数据：

| 数据类别 | 说明 |
|---|---|
| 你配置的翻译源凭证 | LLM API Key / BaseURL、传统翻译源 API Key / 端点 |
| 待翻译文本 | 你划词选中的文本，按需透传给翻译源 |
| 插件设置 | 目标语言、Prompt 偏好、当前生效源等 |

## 2. 你提供的数据

| 数据类型 | 用途 | 存储位置 |
|----------|------|----------|
| LLM API Key / BaseURL | 调用你配置的大模型接口 | 仅本地 `chrome.storage.local`，不上传 |
| 传统翻译源 API Key / 端点（Google / 微软） | 调用你配置的传统翻译官方接口 | 仅本地 `chrome.storage.local`，不上传 |
| 选中的待翻译文本 | 发送给当前生效的翻译源完成翻译 | 不在本插件留存，仅按需透传 |
| 插件设置（目标语言、Prompt、生效源等） | 记忆你的偏好 | 仅本地 `chrome.storage.local` |

## 3. 数据流向

待翻译文本会发送给**当前生效的翻译源**，具体取决于你的配置：

1. **你配置并启用的 LLM 提供方**（云端 OpenAI 兼容接口 / 原生 Anthropic Messages API 端点 / 本地 Ollama）：文本发送给该 LLM 接口。若启用本地模型，文本仅在本机流转，不外传；若启用云端模型（含原生 Anthropic 端点，如 Claude 官方），文本按该云服务商隐私政策处理。
2. **你配置并启用的传统翻译源**（Google / 微软，填有自有 API Key）：文本发送给该官方翻译 API。
3. **内置免 Key 免费翻译源**（**v0.2 新增**）：插件内置 Google 翻译、微软翻译的免 Key 公共端点，作为**用户可选的免费翻译源**。全新安装、用户未做选择时，**默认选中微软翻译**（显式默认值，保证开箱即用），用户可随时在配置页切换到 Google 或自有源。这不是「兜底」机制——源失败时不自动切换到其它源，仅返回错误提示，由用户人工切换。

> ⚠️ **默认外传提示**：由于全新安装默认选中微软翻译免费源，你在未做任何配置时划词翻译，文本会**默认外传**给微软。若选择 Google 免费源，文本会外传给 Google。若不希望文本外传给这些服务，请在配置页切换到自有源（如本地 Ollama，文本仅在本机流转）。

本插件**不会**将你的文本或 Key 发送给插件作者或任何与本插件无关的第三方。文本只发给上述当前生效的翻译源。

## 4. 数据安全

- 所有 API Key（LLM 与传统翻译源）仅存于浏览器本地存储（`chrome.storage.local`），不出现在日志、错误上报或 commit 中。
- 建议用户妥善保管 Key，定期轮换。
- 限制说明：没有任何系统能做到 100% 安全；翻译源服务方的数据处理不在本插件控制范围内。

## 5. 第三方服务

本插件本身的翻译能力由你配置或插件内置的翻译源提供：

| 第三方服务 | 接入方式 | 说明 |
|---|---|---|
| LLM 提供方（你配置） | 你填入的云端 OpenAI 兼容接口 / 原生 Anthropic Messages API 端点（如 Claude 官方）/ 本地 Ollama | 隐私政策独立适用 |
| Google 翻译 | 内置免 Key 公共端点（免费可选源）/ 你填入的官方 API Key | 免 Key 公共端点为**非官方端点**，稳定性与隐私处理以 Google 政策为准；Google 翻译在中国大陆不可达 |
| 微软翻译 | 内置免 Key 公共端点（免费可选源，默认选中）/ 你填入的官方 API Key | 免 Key 公共端点为**非官方端点**，稳定性与隐私处理以微软政策为准 |

`[⚠️ 需要法律复核]` 第三方服务所在地可能位于用户司法辖区之外，数据出境与处理责任由相应服务方承担，建议用户自行评估其隐私条款。

## 6. 数据保留

- 本插件**不在本地留存**待翻译文本，翻译完成后不持久化译文历史（v0.2 范围内不包含历史记录功能）。
- 你的配置与设置保留至你删除或卸载插件。

## 7. 用户权利

- **访问与删除**：你可在浏览器扩展管理中卸载本插件，或在选项页清除配置，所有本地数据随即删除。
- **控制数据流向**：你可通过配置页选择翻译源；配置本地模型（如 Ollama）可使文本不外传；全新安装默认选中微软翻译免费源，你可随时切换到 Google 免费源或自有源。
- `[⚠️ 需要法律复核]` 针对适用司法辖区（如 GDPR/CCPA）的完整数据主体权利（访问、删除、更正、限制处理、数据可携、拒绝、投诉）及其行使流程，待法律复核后补充。

## 8. 合规说明

- 不涉及 Cookies、广告追踪。
- 不面向未成年人收集数据。
- 不出售用户数据。
- 涉及云端或第三方翻译源传输时，数据出境与处理责任由用户选择的服务方承担。
