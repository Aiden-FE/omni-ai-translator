# 可访问性 — LLM Translator

## 对比度

- 浮层深色背景（#1F2937）配浅色文字（#F9FAFB），对比度满足 WCAG AA。
- 错误/成功色用于辅助信息，不作为唯一区分手段（同时有文字"❌"/"✅"）。

## 键盘

- 设置页所有可操作元素（输入框、按钮、下拉）可通过 Tab 聚焦与操作。
- 浮层为鼠标交互产物，第一版不强求键盘可达；后续可加快捷键。

## 文字

- 不依赖纯图标传达信息，关键状态均有文字说明。
- 输入框带 placeholder 说明用途。

## 本地化

- 第一版界面文案为中文；目标语言由用户配置，支持任意语言输入。
- 后续可抽取文案做 i18n。
