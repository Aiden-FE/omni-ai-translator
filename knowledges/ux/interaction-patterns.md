# 交互模式 — LLM Translator

## 划词翻译（核心，v0.1）

### 触发

- 用户在网页选中文本 → `mouseup` 释放
- 选区为空或超 5000 字符时不触发

### 流程

```
选中释放 → 浮层出现在选区右下方（pageX+8, pageY+8）
  → 显示"翻译中…"
  → 后台调用 LLM
  → 成功：显示译文
  → 失败：显示 "❌ <错误信息>"
```

### 关闭

- 点击浮层外任意位置 → 关闭
- 新选区触发 → 旧浮层移除

### 状态反馈

| 状态 | 反馈 |
|------|------|
| 加载 | "翻译中…" |
| 成功 | 译文文本 |
| 失败 | "❌ " + 错误（如未配置提供方、网络错误、HTTP 错误码） |

## 设置页（options）

### 提供方管理

- 添加 / 删除 / 编辑提供方
- 启用：单选启用一个提供方为当前翻译后端
- 测试连通：发送 "hello→中文" 最小请求，显示译文或错误

### 默认目标语言

- 文本输入，自由填写（如「中文」「English」），传入 Prompt

## 弹窗（popup）

- 轻量入口，说明使用方式 + 「打开设置」按钮
- 第一版不承载复杂交互

## 待规划交互（后续版本）

- 划词翻译浮层：重译、复制译文、切换目标语言（v0.1 暂不暴露，先验证核心）
- 快捷键触发
- 小窗输入翻译窗口
