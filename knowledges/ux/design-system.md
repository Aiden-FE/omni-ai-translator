# 设计系统 — LLM Translator

> **v0.3 主题改造方向（2026-07-06 规划）**：v0.3 上线前全面美化应用，引入 shadcn/ui + tailwindcss 重构 UI。当前色彩为灰白黑冷色调（见下表），将改为**适度明亮鲜艳、不抢眼**的主题——弃用纯灰白黑冷色，引入有彩色作主色 / 强调色，提升吸引力但保持阅读舒适。具体色板与设计 token 待 v0.3 UX 规格阶段（`prodflow-ux-spec` / `web-design-engineer`）产出，届时下表冷色调将被替换。

## 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 浮层背景 | `#1F2937` | 深灰，强对比保证可读 |
| 浮层文字 | `#F9FAFB` | 浅色文字 |
| 主操作按钮 | `#1F2937` | 启用态深色填充 |
| 错误 | `#DC2626` | 失败提示 |
| 成功 | `#16A34A` | 成功提示 |
| 页面边框 | `#E5E7EB` / `#D1D5DB` | 卡片与输入框 |

## 字体

- 系统字体栈：`system-ui, -apple-system, sans-serif`
- 正文 14px / 行高 1.5
- 标题 16px

## 间距

- 基准 4px，常用 8px / 12px / 24px
- 浮层内边距 10px 12px
- 卡片内边距 12px

## 圆角

- 卡片 / 浮层：8px
- 输入框 / 按钮：4px

## 组件

### 翻译浮层（content）

- 绝对定位，z-index 最高（2147483647）
- 最大宽度 360px，自动换行
- 暗色背景 + 阴影

### 触发按钮（content）

- 圆形 24×24px，背景 #1F2937，hover #374151
- 居中显示「译」字，13px 600 字重
- 划词后出现，点击触发翻译

### 提供方卡片（options）

- 边框 + 圆角，纵向排列
- 行内排列输入框与操作按钮

## 响应式

- 弹窗固定 240px 宽
- 选项页最大 720px 居中
- 浮层宽度自适应选区与内容，不溢出视口
