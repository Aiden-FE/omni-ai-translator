## 详细描述

```
Omni AI Translator 是一款浏览器划词翻译插件，让你在阅读外文网页时即时翻译选中文本。

核心能力：

• 划词翻译 — 选中网页任意文本，点击「译」按钮即可翻译，译文在浮层中即时展示，无需跳转页面。

• 免 Key 开箱即用 — 内置免费翻译源（Google 翻译、微软翻译），安装后直接使用，无需注册或申请 API Key。

• 自带 Key 接入云端大模型 — 支持 OpenAI 兼容接口、Anthropic Claude 等主流大模型，填入你自己的 API Key 即可使用更高质量的 AI 翻译。

• 本地模型支持 — 支持 Ollama 本地模型接口，完全离线翻译，数据不离开你的设备。

• 隐私可控 — API Key 仅存储在本地浏览器中，不上传至任何服务器。使用本地模型时，翻译请求不经过外部网络。

• 多翻译源切换 — 可同时配置多个翻译源，随时切换，满足不同场景的质量与速度需求。

适合需要在浏览外文内容时快速翻译的用户，无论是阅读技术文档、新闻资讯还是学术论文，选中即译，不打断阅读节奏。
```
