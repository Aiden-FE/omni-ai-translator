# 合规一致性交叉核对清单

## 核对方法

| 材料 | 表述 | 一致性 |
|---|---|---|
| PRIVACY-POLICY.md 第 1 节 | 不收集任何个人身份信息 | 基准 |
| 隐私实践表 | 身份信息=否 | 一致 |
| 数据用途披露 第 1 节 | 不收集 PII | 一致 |

| 材料 | 表述 | 一致性 |
