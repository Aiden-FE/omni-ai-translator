# 数据用途披露 (Data Disclosure)

> 关联 Issue: #64 | PRD Issue: #52 | 版本: v0.3 | 创建日期 2026-07-10
> 事实来源: knowledges/product-wiki/privacy/PRIVACY-POLICY.md

## 概述

LLM Translator 是一款基于 LLM 的 Chrome 浏览器翻译插件，允许用户配置翻译源(云端大模型、本地大模型、传统翻译接口)，提供划词翻译能力。本文档披露插件如何处理用户数据，与隐私实践表和 PRIVACY-POLICY.md 保持一致。

## 1. 数据收集

本插件**不收集**以下数据:
- 个人身份信息(PII)
- 认证信息(API Key 仅存本地，不上传、不收集)
- 位置信息
- 浏览历史
- 用户活动数据(不内置分析/追踪 SDK)
- 网站内容(content script 仅读取用户主动选中的文本)
- Cookie(不使用 Cookie 追踪)

插件在本地处理的数据:
| 数据类别 | 说明 |
|---|---|
| 翻译源凭证 | LLM API Key / BaseURL、传统翻译源 API Key / 端点 |
| 待翻译文本 | 用户划词选中的文本，按需透传给翻译源 |
| 插件设置 | 目标语言、Prompt 偏好、当前生效源等 |

## 2. 数据使用

- **待翻译文本**: 按需透传给当前生效的翻译源完成翻译。翻译完成后不在本地留存，不持久化译文历史。
- **API Key**: 仅用于调用用户配置的翻译接口，存于本地不上传。
- **插件设置**: 记忆用户偏好(目标语言、生效源等)，存于本地。

## 3. 数据共享

待翻译文本发送给**当前生效的翻译源**，具体取决于用户配置:

1. **云端 LLM 提供方**(用户配置): 文本发送给该 LLM 接口(OpenAI 兼容接口 / 原生 Anthropic Messages API 端点 / 其他云端模型)
2. **传统翻译源**(用户配置自有 Key): 文本发送给 Google 或微软官方翻译 API
3. **内置免 Key 免费源**:
   - 微软翻译(默认选中): 全新安装默认选中，文本外传给微软
   - Google 翻译: 用户可选，文本外传给 Google
4. **本地模型**(Ollama): 文本仅在本机流转，不外传

本插件**不会**将文本或 API Key 发送给插件作者或任何与本插件无关的第三方。

## 4. 数据存储

| 数据类型 | 存储位置 | 持久化 |
|---|---|---|
| LLM API Key / BaseURL | chrome.storage.local | 保留至卸载 |
| 传统翻译源 API Key / 端点 | chrome.storage.local | 保留至卸载 |
| 插件设置 | chrome.storage.local | 保留至卸载 |
| 待翻译文本 | 内存(仅翻译过程中) | 不持久化，翻译完成后不留存 |

本插件不在本地留存待翻译文本，翻译完成后不持久化译文历史(v0.3 范围内不含历史记录功能)。

## 5. 数据安全

- 所有 API Key 仅存于浏览器本地存储(chrome.storage.local)，不出现在日志、错误上报或代码提交中。
- 建议用户妥善保管 Key，定期轮换。
- 限制说明: 没有任何系统能做到 100% 安全;翻译源服务方的数据处理不在本插件控制范围内。

## 6. 用户权利

- **访问与删除**: 用户可在浏览器扩展管理中卸载本插件，或在选项页清除配置，所有本地数据随即删除。
- **控制数据流向**: 用户可通过配置页选择翻译源:
  - 配置本地模型(Ollama)可使文本不外传
  - 选择 Google/微软免费源时文本外传给对应服务商
  - 填入自有 API Key 时文本发送给对应官方 API
  - 全新安装默认选中微软翻译免费源，用户可随时切换

> 注: 针对适用司法辖区(如 GDPR/CCPA)的完整数据主体权利及其行使流程，待法律复核后补充。
